# PSD 自动填充配置与使用手册

本文档旨在帮助**管理员**和**普通用户**快速掌握“PSD 自动填充”功能的使用方法。哪怕你是第一次接触，跟着步骤做也能轻松搞定！

---

## 一、 什么是“PSD 自动填充”？

简单来说，就是把 Excel 表格里的数据（如商品名、价格、主图），自动填进 PSD 模版的对应位置，一键生成大量图片，不再需要手动一个个改。

*   **管理员**：负责定“规则”（哪个格子填什么数据）。
*   **用户**：负责传“数据”（上传 Excel 并点击生成）。

---

## 二、 管理员配置篇 (如何制作模版)

作为管理员，你需要告诉系统：PSD 里的每一个图层，应该对应 Excel 表格里的哪一列。

### 第一步：进入配置页面
1.  在模版列表找到你上传的 PSD 模版。
2.  点击 **“配置商品位”** (或类似的设置图标) 进入编辑器。

### 第二步：上传参考表格 (定义“字典”)
系统需要知道你的 Excel 表格里有哪些表头（比如“标题”、“售价”、“主图链接”）。
1.  在左侧侧边栏，找到 **“表格字段配置”** 区域。
2.  点击 **“上传表格”** 按钮。
    *   也可以直接把 Excel 文件拖到这个区域，松开就会自动上传。
3.  选择一份标准的 Excel 文件上传（只需包含表头即可，数据行不重要）。
4.  上传成功后，你会看到系统识别出的所有字段名。
    *   *小技巧：如果有些字段不需要，可以点击“删除”隐藏它，避免干扰。*

### 第三步：创建“商品位” (定义“坑位”)
如果一张图里要放 3 个商品，就创建 3 个商品位。通常只需要 1 个。
1.  在左侧侧边栏，找到 **“商品位列表”**。
2.  点击右上角的 **“+”** 号，创建一个新商品位（默认叫“商品位 1”）。
3.  点击选中这个商品位。

### 第四步：选择变量并加入商品位
1.  在中间的 **画布** 上，点击你想要动态修改的图层（比如原本写着“¥99”的文字图层，或者原本是商品图的图片图层）。
    *   *支持多选：按住 Shift 键或在画布上拖拽框选，可以同时选中多个图层。*
2.  选中后，画布上方会出现一个悬浮条，显示“已选中 X 个变量”。
3.  点击悬浮条上的 **“添加到 商品位 1”** 按钮。
4.  此时，左侧侧边栏的“商品位 1”下方，就会列出你刚才添加的变量。

### 第五步：绑定规则 (连线)
这是最关键的一步！你需要把图层和表格字段对应起来。
1.  在左侧“商品位 1”的列表里，找到刚才添加的变量（比如“价格图层”）。
2.  点击它下方的 **“绑定字段”** 下拉框。
3.  在下拉列表中，选择对应的 Excel 表头（比如“售价”）。
    *   *意思就是：以后 Excel 里“售价”这一列填什么，这个图层就显示什么。*
4.  **进阶设置**：如果你想让数据变得更聪明（比如自动加前缀、自动判断是否偏光），请点击变量右侧的 **“规则”按钮**（小魔法棒图标），详细说明见下文“三、 进阶玩法”。

### 第六步：保存
配置完成后，别忘了点击左下角的 **“保存配置”** 按钮。

## 三、 进阶玩法：让数据自动变身 (绑定规则详解)

有时候 Excel 里的原始数据不能直接用，需要“加工”一下。比如表格里写的是 `99`，图片上要显示 `¥99.00`。这就需要用到**绑定规则**。

在变量列表里点击 **“规则”按钮**，可以看到以下几种高级玩法：

### 1. 傻瓜拼句（模板字符串）
你可以把它当成“写一句话”，系统会把句子里标记的字段替换成 Excel 的值。

*   **怎么用（最简单）**：
    1.  在“规则类型”里选择 **“傻瓜拼句（点字段插入）”**
    2.  在上方“字段搜索”里输入关键词找到字段
    3.  直接点击字段按钮，会自动插入到输入框里（不用手写花括号）
    4.  看下面的“预览”是否符合你想要的结果
*   **例子**：
    *   你输入：`款号{{款号}}-{{色号}}`
    *   Excel：款号=`BL3209`，色号=`C90`
    *   **最终图片显示**：`款号BL3209-C90`

### 2. 字段拼接（最常用）
把多个字段的内容连起来，还能加前缀后缀（比如价格加 `¥`）。

| 配置项 | 说明 | 举例 |
| :--- | :--- | :--- |
| **字段 1** | 选择第一个要显示的字段 | 品牌 (Nike) |
| **字段 2** | 选择第二个要显示的字段 | 型号 (Air Max) |
| **中间符号** | 两个字段中间加什么符号 | 空格 |
| **前缀/后缀** | 在字段前后自动加字 | 比如给价格加“¥” |

*   **配置完的效果**：
    *   Excel 数据：品牌=`Nike`，型号=`Air Max`
    *   **最终图片显示**：`Nike Air Max`
    *   *注：如果某个字段为空，会自动忽略，不会显示多余的连接符。*

### 3. 特殊值转换 - 关键字判断 (智能文案)
根据表格里有没有某个词，自动显示不同的文案。常用于“偏光/非偏光”判断。

| 配置项 | 说明 | 举例 |
| :--- | :--- | :--- |
| **来源字段** | 检查哪个字段的内容 | 镜片功能 |
| **转换模式** | 选择“关键字包含” | - |
| **关键字** | 只要包含这个字就算命中 | 偏光 |
| **命中文案** | 包含关键字时显示什么 | 高清偏光镜片 |
| **未命中文案** | **不**包含关键字时显示什么 | 非偏光镜片 |

*   **配置完的效果**：
    *   Excel 数据：镜片功能=`TAC偏光` -> **最终显示**：`高清偏光镜片`
    *   Excel 数据：镜片功能=`PC镜片` -> **最终显示**：`非偏光镜片`

### 4. 特殊值转换 - 值映射表 (查字典)
把表格里的“代码”自动翻译成“人话”。

| 配置项 | 说明 | 举例 |
| :--- | :--- | :--- |
| **来源字段** | 检查哪个字段的内容 | 镜框材质 |
| **转换模式** | 选择“值映射表” | - |
| **映射 1** | 当 Excel 是 A 时，显示 B | `TR` -> `高性能尼龙` |
| **映射 2** | 当 Excel 是 C 时，显示 D | `/` -> `无边框` |

*   **配置完的效果**：
    *   Excel 数据：镜框材质=`TR` -> **最终显示**：`高性能尼龙`
    *   Excel 数据：镜框材质=`METAL` (未定义映射) -> **最终显示**：`METAL` (保持原样)

### 5. 镜片类型汇总 (行业专用)
专门用于眼镜行业，自动根据“物料号”和“镜片类型”生成汇总文案。

*   **配置完的效果**：
    *   自动分析同款商品的所有镜片，输出如 `高清偏光/非偏光` 或 `高清偏光`。

### 6. 分段拼接（高级玩家）
如果你需要“某段为空就整段隐藏、并且不显示中间符号”，可以用两种方式做：
*   **优先推荐**：用“字段拼接”，把不需要的内容分到前缀/后缀里，并保持 Excel 数据干净。
*   **更复杂的情况**：用“高级（原始 JSON）”，让工程同事帮你把规则写好（适合规则非常多、非常复杂的模版）。

## 四、 用户使用篇 (如何生成图片)

作为用户，你只需要准备好 Excel 表格，就能一键生成图片。

### 第一步：选择模版
1.  在工作台首页，点击 **“PSD 自动填充”** 标签页。
2.  在模版列表中，选择一个已配置好的模版。

### 第二步：上传数据
1.  在右侧的 **“数据绑定”** 面板中，点击 **“上传 Excel”**。
2.  选择你整理好的商品数据表格。
    *   *注意：表格的表头名称最好和管理员配置的一致，系统会自动匹配。如果不一致也不要紧，后面可以手动选。*

### 第三步：绑定数据与填充
1.  上传后，你会看到表格数据列表。
2.  **选中一行数据**：点击表格里的某一行（比如“商品 A”那一行）。
3.  **选择商品位**：在表格上方的下拉框里，选择 **“商品位 1”**（或其他位置）。
4.  点击 **“添加到商品位”** 按钮。
    *   *此时，系统建立了“这一行数据” -> “这个商品位”的关联。*
5.  点击 **“自动填充”** 按钮（通常在商品位列表附近）。
    *   *见证奇迹的时刻：左侧画布上的文字和图片，会瞬间变成你 Excel 里对应的内容！*

### 第四步：微调与导出
1.  检查画布上的效果。如果图片位置不对，可以直接在画布上拖拽调整；如果文字太长，可以直接双击修改。
2.  确认无误后，点击 **“导出结果”**。
3.  系统会生成最终的 PSD 或图片供你下载。

## 五、 常见问题 (FAQ)

**Q: 为什么上传 Excel 后，自动填充没反应？**
A: 请检查以下几点：
1.  是否点击了“添加到商品位”？（必须先建立行与商品位的关联）。
2.  管理员是否正确绑定了字段？（如果管理员没绑定“售价”字段，Excel 里的售价通过去）。
3.  Excel 表头是否匹配？（虽然系统支持模糊匹配，但最好保持一致）。

**Q: 后台删除了某个变量，为什么前台还能看到？**
A: 系统有缓存机制，请尝试刷新页面。新版本中，后台删除（隐藏）的变量，前台会自动过滤，不再显示，也不会遮挡其他操作。

**Q: 我手动改了画布上的字，再点自动填充会被覆盖吗？**
A: 会的。自动填充以 Excel 数据为准。如果你想保留手动修改，请在自动填充**之后**再手动改，或者不要对该位置进行字段绑定。

**Q: 图片支持什么格式？**
A: Excel 里填写的图片链接，支持 `http/https` 开头的远程链接，也支持 Base64 格式。推荐使用直链。

**Q: 导出 PSD 失败，提示“文档数据超过 2GB 限制/常规 Photoshop 错误”怎么办？**
A: 这是 PSD 格式的已知限制：当导出的文件数据超过 2GB 时，Photoshop 无法保存为 `.psd`。
1. 系统会优先自动降级导出为 **`.psb`（大文档格式）** 并提供下载；`.psb` 可直接用 Photoshop 打开且支持超大文件。
2. 如果当前 Photoshop 不支持导出 PSB，系统会改为导出 **“扁平化 PSD”**（只保留单图层），确保你至少能拿到可打开的 PSD 文件。
3. 如需保留完整图层结构，请联系管理员升级 Photoshop 版本后再导出。
