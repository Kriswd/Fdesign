# Fdesign Roadmap

## Now

- 让 README 首屏更清楚地表达“Excel 商品数据 -> 批量 PSD 成品”的生产价值。
- 补齐 Demo、发布素材、Issue 模板和公开讨论入口。
- 建立 V3.0.0 Release，给外部分享一个稳定落点。
- 收集第一批真实电商模板、导出失败和文档困惑。

## Next

- 增加更多开源可复现的模板配置示例。
- 补充从空白 PSD 到批量导出的完整图文教程。
- 提炼常见 Photoshop 导出失败的排查清单。
- 继续改进短视频和排障案例，让非开发者更快完成第一次试跑。

## Later

- 探索跨模板任务模板复用。
- 优化大批量导出性能与失败恢复体验。
- 拓展更多电商平台命名规则和素材匹配策略。
- 沉淀社区提交的模板案例和最佳实践。

## Good First Contributions

- 改进 README 的英文摘要和术语。
- 提交一个不含隐私和版权风险的模板使用案例说明。
- 给 `docs/DEMO.md` 补充截图标注或更清楚的操作步骤。
- 复现并整理一个 Photoshop 导出失败的最小案例。
- 修正运行文档中不清楚的 Windows/Node/Photoshop 环境说明。

## How We Decide Priority

优先处理能让真实电商图像生产更稳定、更容易上手、更容易复用的问题。排序时会看三个维度：是否影响导出结果、是否阻碍新用户快速启动、是否能帮助更多模板场景复用。
