# 导出参考线冲突：变量定位高亮与配置防呆 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 让 DUP_IMAGE_GUIDE_MISMATCH 可定位、可高亮、可预防，并下调 PSD 自动填充默认缩放。

**Architecture:** 后端透传冲突结构与明细；前端在用户端/管理端统一实现“attention 高亮”能力；管理端保存前做同字段 guidePick 一致性校验。

**Tech Stack:** Node/Express, React, TailwindCSS, Vite
