# 画板命名规则升级与导出默认折叠 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 让 PSD 自动填充导出在所有画板上按用户上传图文件名规则重命名，并在导出结果中默认折叠所有画板图层组。

**Architecture:** 命名规则收敛在 `server/utils/templateMeta.js`（由 `buildArtboardRenameMap` 统一产出 `artboardRenames`），执行端在 `server/photoshop/render_export.jsx` 继续按图层所属画板应用重命名；新增“导出前折叠画板组”步骤，挂在 `runSingle/runBatch/runPsdBundle` 的导出前时机，保证所有导出模式一致。测试采用 Node 字符串契约 + 纯函数断言 + 手工 PSD 验证三层闭环。

**Tech Stack:** Node.js（ESM）、Express 服务层、ExtendScript JSX（ES3）。

const STYLE_RE = /([A-Za-z]{2}\d{4})/;
const STYLE_COLOR_RE = /([A-Za-z]{2}\d{4})[\s_-]*([A-Za-z]\d{2})/;

function parseStyleAndColor(baseName) { ... } // 返回 { styleNo, styleColor }
function rewriteTmallMainArtboardName(artboardName, styleNo) { ... } // 天猫主图 ... -> 只替换中间款号段
function isStyleColorArtboardName(name) { ... } // 例如 BL3208 C50

function buildArtboardNextName(artboard, baseName) {
  const parsed = parseStyleAndColor(baseName);
  if (!parsed.styleNo) return artboard;
  if (isTmallMainArtboardName(artboard)) return rewriteTmallMainArtboardName(artboard, parsed.styleNo);
  if (isStyleColorArtboardName(artboard) && parsed.styleColor) return parsed.styleColor;
  return parsed.styleColor || parsed.styleNo;
}
```

function collapseAllArtboardGroups(workDoc, logArr) {
  try {
    app.activeDocument = workDoc;
    executeAction(stringIDToTypeID("collapseAllGroupsEvent"), undefined, DialogModes.NO);
    if (logArr) logArr.push("artboard_groups_collapsed=true");
  } catch (e) {
    if (logArr) logArr.push("artboard_groups_collapsed=false:" + String(e));
  }
}
```

- `runSingle`：`applyUpdatesToDoc` 后、`exportImage` 前

- `runPsdBundle`：最终 `exportImage` 前
